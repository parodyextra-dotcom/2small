# 🏍️ 2종소형면허 회원관리 시스템

## 📋 프로젝트 개요
2종소형면허 교육 회원을 관리하기 위한 모바일 최적화 웹 애플리케이션입니다.
회원 등록/수정/삭제, 전화/문자 발신, 통계 차트, 데이터 백업/복원 등의 기능을 제공합니다.

## ✅ 구현 완료된 기능

### 1. 로그인 시스템
- 비밀번호 기반 접속 제한 (비밀번호: `2026한성`)
- 세션 스토리지 기반 로그인 유지

### 2. 회원 관리
- **회원 목록** - 카드형 목록 보기, 검색(이름/전화번호), 상태/성별 필터링
- **회원 등록** - 이름, 나이, 성별, 전화번호, 검정일자, 성격, 진행상태, 특이사항 입력
- **회원 수정** - 기존 회원 정보 수정
- **회원 삭제** - 확인 모달 후 삭제
- **회원 상세보기** - 전체 정보 카드 뷰

### 3. 연락 기능
- **전화 걸기** - `tel:` 링크로 전화앱 연동
- **개별 문자** - SMS 앱 연동 (iOS/Android 호환)
- **단체 문자** - 상태별 수신자 선택 + 일괄 문자 발송
- **빠른 템플릿** - 수업안내, 시험일정, 합격축하, 일정변경, 준비물안내

### 4. 통계 & 차트
- 일일/주별/월별/년별 활동 통계
- Bar Chart (활동 추이)
- Doughnut Chart (활동 비율)
- 활동 기록 로그

### 5. 데이터 관리 (⭐ 핵심 기능)
- **로컬스토리지 저장** - 신규 회원은 브라우저 로컬스토리지에 자동 저장
- **p1.json 불러오기** - 서버의 p1.json 파일에서 회원 데이터 가져오기
- **JSON 파일 불러오기** - 기기에서 JSON 파일 선택하여 데이터 임포트
- **JSON으로 내보내기** - 현재 회원 데이터를 JSON 파일로 다운로드
- **전체 데이터 삭제** - 로컬스토리지 초기화
- **중복 체크** - 이름+연락처 기준 중복 데이터 자동 제외

## 📁 파일 구조

```
index.html          - 메인 애플리케이션 (HTML + CSS + JS 단일 파일)
p1.json             - 초기 회원 데이터 (35명, 검정일자/성명/나이/성별/연락처/판정)
p1.json.txt         - 원본 데이터 파일
index_original.html - 원본 HTML 파일 (백업)
README.md           - 프로젝트 문서
```

## 💾 데이터 저장 구조

### LocalStorage 키
| 키 | 설명 |
|----|------|
| `members_data` | 회원 데이터 배열 (JSON) |
| `activity_logs` | 활동 기록 배열 (JSON, 최대 500개) |

### 회원 데이터 필드
```json
{
  "id": "고유ID",
  "name": "성명",
  "age": 34,
  "gender": "남/여",
  "phone": "010-0000-0000",
  "status": "합격/불합격/교육전/교육중/시험대기",
  "exam_date": "2025-01-04",
  "personality": "성격유형",
  "notes": "특이사항",
  "reg_date": "등록일(ISO)",
  "seq": 1
}
```

### p1.json 데이터 형식 (호환)
```json
{
  "연번": 1,
  "검정일자": "2025-01-04",
  "성명": "홍길동",
  "나이": 34,
  "성별": "남",
  "연락처": "010-0000-0000",
  "판정": "합격"
}
```

## 🛠️ 사용된 기술

- **HTML5 / CSS3 / Vanilla JavaScript** - 프레임워크 없이 구현
- **LocalStorage** - 클라이언트 데이터 저장
- **Font Awesome 6.4** - 아이콘
- **Noto Sans KR** - 한국어 폰트
- **Chart.js** - 통계 차트

## 📱 사용 방법

1. 웹사이트 접속 → 비밀번호 입력 (`2026한성`)
2. 헤더의 **[데이터관리]** 버튼 클릭
3. **[p1.json에서 불러오기]** 선택 → 35명의 초기 데이터 로드
4. 회원 목록에서 관리 시작!
