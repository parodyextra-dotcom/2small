# 🏍️ 2종소형면허 회원관리 앱

2종소형면허 교육생 회원을 체계적으로 관리하고, 개별/단체 문자 발송 기능을 제공하는 웹 앱입니다.

## ✅ 구현된 기능

### 회원 관리 (CRUD)
- **회원 등록**: 이름, 나이, 성별, 전화번호, 성격, 진행상태, 특이사항 입력
- **회원 목록**: 카드형 목록으로 한눈에 회원 현황 파악
- **회원 상세**: 개별 회원의 모든 정보 확인
- **회원 수정**: 등록된 정보 수정
- **회원 삭제**: 확인 모달 후 삭제

### 검색 & 필터
- **이름/전화번호 검색**: 실시간 검색
- **상태별 필터**: 교육전, 교육중, 시험대기, 합격, 불합격
- **성별 필터**: 남성, 여성

### 문자 발송
- **개별 문자**: 회원별 개별 문자 전송 (기기 문자앱 연동)
- **단체 문자**: 다수 회원 선택하여 일괄 문자 전송
- **빠른 템플릿**: 수업안내, 시험일정, 합격축하, 일정변경, 준비물안내
- **상태별 일괄 선택**: 교육중, 시험대기 등 상태별 빠른 선택

### 전화 기능
- **바로 전화걸기**: 목록/상세에서 즉시 전화 가능

### 대시보드
- **실시간 통계**: 전체 회원수, 교육중, 합격자 수 표시

## 📱 페이지 구조

| 경로 | 설명 |
|------|------|
| `index.html` | 메인 앱 (SPA, 모든 기능 포함) |

### 앱 내부 페이지 (탭 네비게이션)
- **회원목록**: 전체 회원 카드 리스트 + 검색/필터
- **회원등록**: 신규 회원 등록 폼
- **단체문자**: 다수 수신자 선택 + 문자 발송
- **회원상세**: 개별 회원 정보 + 전화/문자/수정/삭제
- **개별문자**: 특정 회원에게 문자 보내기

## 🗄️ 데이터 모델

| 필드 | 타입 | 설명 |
