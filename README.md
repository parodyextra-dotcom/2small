# 🏍️ 2종소형면허 회원관리 앱

2종소형면허 교육생 회원을 체계적으로 관리하고, 개별/단체 문자 발송 및 통계 분석 기능을 제공하는 웹 앱입니다.

## ✅ 구현된 기능

### 회원 관리 (CRUD)
- **회원 등록**: 이름, 나이, 성별, 전화번호, 성격, 진행상태, 특이사항 입력
- **회원 목록**: 카드형 목록으로 한눈에 회원 현황 파악
- **회원 상세**: 개별 회원의 모든 정보 확인
- **회원 수정**: 등록된 정보 수정
- **회원 삭제**: 확인 모달 후 삭제

### 검색 & 필터
- **이름/전화번호 검색**: 실시간 검색
- **상태별 필터**: 교육전, 교육중, 시험대기, 합격, 불합격
- **성별 필터**: 남성, 여성

### 문자 발송
- **개별 문자**: 회원별 1:1 문자 전송 (기기 문자앱 연동)
- **단체 문자**: 다수 회원 선택하여 일괄 문자 전송
- **빠른 템플릿**: 수업안내, 시험일정, 합격축하, 일정변경, 준비물안내
- **상태별 일괄 선택**: 교육중, 시험대기 등 상태별 빠른 선택

### 📊 통계 (NEW)
- **기간별 조회**: 일일 / 주별 / 월별 / 년별 전환
- **날짜 네비게이션**: 이전/다음으로 기간 탐색
- **요약 카드**: 전화 발신, 문자 발송, 신규 회원 수 표시
- **막대 차트**: 기간별 전화/문자/신규회원 추이 (Chart.js)
- **도넛 차트**: 활동 유형별 비율 시각화
- **최근 활동 기록**: 최근 20건의 활동 로그 타임라인
- **자동 기록**: 전화/문자/회원등록 시 자동으로 활동 로그 저장

### 전화 기능
- **바로 전화걸기**: 목록/상세에서 즉시 전화 가능 (활동 기록 자동 저장)

### 대시보드
- **실시간 통계**: 전체 회원수, 교육중, 합격자 수 표시

## 📱 페이지 구조

| 경로 | 설명 |
|------|------|
| `index.html` | 메인 앱 (SPA, 모든 기능 포함) |

### 앱 내부 페이지 (탭 네비게이션 4개)
1. **회원목록**: 전체 회원 카드 리스트 + 검색/필터
2. **회원등록**: 신규 회원 등록 폼
3. **단체문자**: 다수 수신자 선택 + 문자 발송
4. **통계**: 일일/주별/월별/년별 활동 통계 차트
5. **회원상세** (서브): 개별 회원 정보 + 전화/문자/수정/삭제
6. **개별문자** (서브): 특정 회원에게 문자 보내기

## 🗄️ 데이터 모델

### members 테이블
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| name | text | 이름 |
| age | number | 나이 |
| gender | text | 성별 (남/여) |
| phone | text | 전화번호 |
| personality | text | 성격 |
| notes | rich_text | 특이사항 |
| status | text | 진행상태 (교육전/교육중/시험대기/합격/불합격) |
| reg_date | datetime | 등록일 |

### activity_logs 테이블
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| type | text | 활동 유형 (call/sms/bulk_sms/new_member) |
| member_id | text | 대상 회원 ID |
| member_name | text | 대상 회원 이름 |
| detail | text | 활동 상세 내용 |
| count | number | 대상 인원수 (단체문자 시) |
| log_date | datetime | 활동 일시 |

## 🔧 기술 스택
- **HTML5 + CSS3 + Vanilla JavaScript** (단일 파일)
- **Chart.js** (통계 차트 시각화)
- **RESTful Table API** (데이터 저장)
- **Font Awesome 6** (아이콘)
- **Noto Sans KR** (한글 폰트)
- **반응형 디자인** (모바일/데스크톱 대응)
